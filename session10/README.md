Se ha introducido un ejemplo de herencia múltiple y como solucionarlo con java + interficies.

clase abstracta Animal y derivadas: Perro, Gato, Pájaro, Mosca
clase abstracta Vehiculo y derivadas: Coche, Moto, Helicoptero, Avioneta.

Animal tiene un método abstracto: habla
Vehiculo tiene métodos abstractos: encender y apagar

Volador es una clase abstracta con el método: volar -> se ha convertido en interfaz y pajaro, mosca, helicoptero y avioneta deber implementar.


Se ha introducido Controller como orquestador de la aplicación y dejar el main vacío.

Se ha introducido el concepto de inyeccion de dependencias y controller está sólamente acoplado a interfaces.
