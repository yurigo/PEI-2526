Se ha pedido crear en grupos de 2/3 personas una presentación sobre JSON.

Se han pedido conceptos como:
tipos de datos de json
sintaxis de json y ejemplos correctos e incorrectos
un ejemplo especifico:
alumno/s con info de nombre, edad, asignaturas.
como trabajar JSON en JAVA.
concepto de serialización y deserialización.
Que es xml y diferencias con json.

En la segunda parte se ha creado un miniproyecto para leer el json en java con GSON.

- se ha leido un alumno con su nombre, edad y aprobado(boolean) y se ha mostrado por consola.
- se ha leido un array de alumnos con sus nombre, edad y aprobado(boolean) y se han mostrado por consola.

pendiente:

- crear un alumno nuevo y añadirlo al array de alumnos, y mostrar el resultado por consola.
- crear un alumno nuevo a partir de datos introducidos por el usuario (Scanner) y añadirlo al array de alumnos, y mostrar el resultado por consola.
- guardar el array de alumnos en un nuevo archivo JSON.
